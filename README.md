# PRECISION Scientific Reports

Automatisierter Workflow für wissenschaftliche Berichte mit Quarto und GitHub Pages.

## Features

- Jupyter Notebooks → HTML/PDF/LaTeX/Word
- Automatische Deployment via GitHub Actions
- Journal-Profile (Springer, Elsevier, Nature, Lancet)
- Zotero-Integration
- Responsive Design

## Quick Start

1. Repository klonen
2. Notebook in `notebooks/` erstellen
3. Pushen → fertig!

GitHub Actions generiert automatisch alle Formate.
